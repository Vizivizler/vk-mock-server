## Установка
```
git clone https://github.com/Vizivizler/vk-mock-server.git
```

##### или

```
Качаем и распаковываем в нужное место
```

Переходим в каталог где лежит проект, пишим команду

```
npm install
```

#### Комнда запуска

```
npm start
```

#### Комнда остановки

```
npm stop
```

Запуск идет с модуля pm2, так же будет знимать порт 10879
